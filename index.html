Data Attributes

In order to customize your PayByGroup you need to add the following data attributes to your pbg_info element.

Tip: Remember to add data attributes in the element in the following way data-purchase-name="Hello world" .

Group Purchase

purchase-image-url:
string URL for the PayByGroup image
purchase-name:
string Name of the product/service for the PayByGroup
purchase-id:
string Your ID used to identify the purchase for your records
purchase-cost:
float Total cost of the purchase
purchase-currency:
string Three letter code of the currency being used (defaults to USD if not specified). No automatic conversions happen, so make sure that your gateway accepts it
purchase-link-url:
string URL for the page of the product/service
purchase-description:
string Description of the purchase


Other Commonly Used Data Attributes

These attributes are used on an as needed basis depending on your product/service.

purchase-start-date:
date If your service needs to store a start date. i.e. Reservation start date
purchase-end-date:
date If your service needs to store an end date. i.e. Reservation end date
purchase-description:
date Date on which the hold expires and the inventory becomes subject to availability 

<div class="pbg_info"
  data-purchase-image-url="https://example.com/picture.png"
  data-purchase-name="Awesome Escape"
  data-purchase-id="ESCAPE-23"
  data-purchase-cost="500.45"
  data-purchase-currency="CAD"
  data-purchase-link-url="http://example.com/property"
  data-purchase-description="Great rental for 3 people with lake view." >
</div>

Sample Implementation

Informational Unit(s)

Add an informational unit using any of the copy and design examples listed in the Design Guidelines on one or more of the following pages of your website:

• Homepage
• Search results page
• Product listing page

You may customize the look and feel to match your site as long as the PayByGroup Logo (See Design Assets) is included whenever the term "PayByGroup" is used.1 We recommend you use the version hosted by PayByGroup to eliminate the need to change your code if the logo is updated.

When clicked, the informational unit displays the lightbox explaining how PayByGroup works, and the only option on the lightbox is to close it and keep browsing.

The key parameter that defines these divs as informational units is data-popup-button-action="close" .


No other data attributes need to be defined for the informational units, though you can customize them if you like.

Option to Use PayByGroup

Place the option to use PayByGroup at the earliest point on your site where the total cost is available - including all taxes, fees, and options. This will typically be on the first page of your checkout process.

Tip: This is where you should be sure to set the data attributes listed above and any additional ones you want to include .
